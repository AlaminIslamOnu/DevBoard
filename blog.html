Question-3: What is event delegation in the DOM?

An event listener can be added to a parent element. Event Delegation is a technique in JavaScript where, instead of adding separate event listeners to multiple child elements, a single event listener is added to the parent element.

Question-4: What is event bubbling in the DOM?

Event Bubbling is an event propagation mechanism in JavaScript, where when an event is triggered on a child element, it also propagates (bubbles up) to the parent element.
